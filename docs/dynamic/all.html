# 获取动态列表

https://api.bilibili.com/x/polymer/web-dynamic/v1/feed/all

请求方式：GET

是否需要登录：是

参数名类型必填内容备注
timezone_offsetstr-480
typestr分类all：全部
video：视频投稿

article：专栏
offsetnum分页偏移量翻页时使用
update_baselinestr更新基线获取新动态时使用
