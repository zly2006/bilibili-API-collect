# 高能进度条

高能进度条反应了在时域上，单位时间内弹幕发送量的变化趋势

并用曲线顶点表示在进度条上，实现可视化

# 获取弹幕趋势顶点列表

https://bvc.bilivideo.com/pbp/data

请求方式：GET

url参数：

参数名类型内容必要性备注
cidnum视频cid必要
aidnum稿件avid非必要
bvidnum稿件bvid非必要

json回复：

根对象：

字段类型内容备注
step_secnum采样间隔时间单位为秒
由视频时长决定
tagstrstr？？？作用尚不明确
eventsobj数据本体
debugstr调试信息json字串

events对象：

字段类型内容备注
defaultarray顶点值列表

events对象中的数组：

项类型内容备注
0num顶点1
nnum顶点(n+1)顶点个数由视频时长和采样时间决定
……num…………

示例：

获取视频cid=3724723的获取弹幕趋势顶点列表

curl -G 'https://api.bilibili.com/pbp/data' \
--data-urlencode 'cid=3724723'
